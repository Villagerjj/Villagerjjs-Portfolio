Hello! This is my portfolio for my various low poly 3D models!

Personal Projects:

These are all projects I did to apply the various skills I have learned, or models I just made for fun.

This character is an NPC I made for an asset pack, this model showcases the gradient texture mapping technique, along with animation ready geometry. The model uses a 43x43 texture, however, it is not fully utilized. the eyes and mouth both use a single 32x32 texture.

This character is a low poly version of Doomguy from the Doom (1993). this model was made for fun, and sadly does not employ any techniques. the model is not animation-ready.

This character is a stylized verison of Samus Aran from the Metroid series. the model showcases gradient texture mapping, and animation-ready geometry, all in a low poly style. the model is 674 triangles, and uses a single 8x32 texture. this model is highly optimized for low memory environments, and could look good on underpowered technology such as the N64, PS1, and DS
